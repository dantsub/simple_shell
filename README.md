# Simple shell



### Pre-requisitos 

- OS Ubuntu 14.04 LTS
- gcc version 4.8.4

### Install

Download or clone repository and compliled.

It should be compiled as follows for better operation.

```
gcc -Wall -Werror -Wextra -pedantic *.c -o hsh

```
